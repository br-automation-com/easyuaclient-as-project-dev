Client program examples with OPC UA function blocks

This section explains the communication between 2 PLC systems (client-server) using several program examples.

The following examples has been developed using 2 ARsim as client-server but it should be compatible with 3rd party server.

Communication is established and process values communicated, e.g. querying the current values of individual variables on the OPC UA server from the OPC UA client or calling methods.

To keep the program examples as simple as possible, only the basic functions will be explained. In practice, for example, error handling is much more extensive.

Structured Text is the only programming language used in the program examples.

Version used

Creating the program examples requires Automation Studio 4.4.1 or later and Automation Runtime C4.41 or later.
